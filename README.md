# Tweetable Basic (mobile version)

## Ruby version

v2.7.3

## Dependencies

- rails
- pg
- devise
- omniauth [facebook | github | google]

## Get started

```bash
rails db:create
```

```bash
rails db:migrate
```

```bash
rails db:seed
```

```bash
rails s -b 0.0.0.0
```
